    Notas de la Versión

    Versión 0.1.0

    Descripción

    Este es el primer lanzamiento oficial del proyecto. Incluye la configuración inicial y la implementación de Git Flow para facilitar el desarrollo colaborativo y la gestión de versiones.

    Nuevas Funcionalidades

    • Configuración Inicial del Proyecto: Se ha configurado el proyecto Python con las dependencias y estructuras básicas necesarias.
    • Implementación de Git Flow: Automatización de la configuración de Git Flow para gestionar ramas y versiones de manera eficiente.

    Mejoras

    • Validación de Ramas: Se ha añadido la validación para verificar la existencia de la rama develop tanto local como remotamente.
    • Reseteo de Ramas: Automatización del reseteo de la rama develop desde origin/main en caso de que ya exista.

    Correcciones

    • Manejo de Errores: Se ha mejorado el manejo de errores al crear la rama develop si ya existe localmente, evitando conflictos y errores en el proceso.

    Adiciones

    • Instalación de Dependencias: En caso de existir un archivo requirements.txt, se utilizan las dependencias listadas en él para la inicialización del proyecto.
    • Configuración de .gitignore: En caso de existir un archivo .gitignore, se utiliza el mismo para la configuración del proyecto.

    Recomendaciones

    • Asegúrate de que todas las dependencias estén actualizadas antes de comenzar a trabajar en el proyecto.
    • Revisa y adapta la configuración de Git Flow según las necesidades específicas de tu equipo y flujo de trabajo.

    Full Changelog: https://github.com/HansBuddenbergBlamey/Setup-Python-Project-Template/commits/0.1.0
